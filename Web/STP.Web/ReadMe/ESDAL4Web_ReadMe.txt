About the ESDAL4 Web Portal
------------------------------
ESDAL means ‘Electronic Service Delivery for Abnormal Loads’. It is a web-based portal for assisted
route planning and automated notification of abnormal load movements.

Dependency List
---------------

	- Oracle Client 19c
	- IIS 10
